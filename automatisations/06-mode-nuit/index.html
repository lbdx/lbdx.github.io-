    Mode nuit

    Objectif : Eteindre les appareils restés allumés lors du couché en scannant un tag NFC

    Matériel utilisé

    • 1 badge NFC pouvant être écrit par Home Assistant
    • une télécommande universelle connectée: Broadlink RM mini.
    • 1 télévision non connectée
    • 1 barre de son Yamaha MusicCast
    • 3 ampoules de marque Wiz positionnées dans le salon et la salle à manger

    Mise en oeuvre

    Lecture du badge NFC

    - id: '1612460915835'
  alias: La balise nuit est analysée
  description: ''
  trigger:
  - platform: tag
    tag_id: d23152c3-01e7-48a7-b82b-ab989ba2c686
  condition: []
  action:
  - service: script.sleep
    data: {}
  mode: single

    Groupe de lampes

    Le groupe permet d'agir sur une liste de lampes en une seule action.

    light:
  - platform: group
    name: Lumières salon
    entities:
      - light.salon
      - light.salon2
      - light.salle_a_manger

    Script éteignant les appareils

    • Le device 42e5549cdbc8b0094de6a194ffcbf2bf correspond à la barre de son.

    • Le script tv_onoff permet d'allumer ou d'éteindre la TV.

    script:
  sleep:
    alias: sleep
    sequence:
    - service: light.turn_off
        data: {}
        entity_id: light.lumieres_salon
    - service: media_player.turn_off
        data: {}
        target:
        device_id: 42e5549cdbc8b0094de6a194ffcbf2bf
    - condition: state
        entity_id: sensor.tv_template
        state: 'on'
    - service: script.tv_onoff
        data: {}
    mode: single
    icon: mdi:sleep
